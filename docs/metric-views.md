# Metric Views

Argus rolls your usage up several ways, each a tab in the app. This page explains
what the numbers in each view mean.

## Activity

The home view: headline totals (sessions, model responses, tokens and estimated
cost), the recommendations Argus surfaces, and trends for tokens and cost over
time broken down by [source](/glossary#source).

## Projects

Your usage grouped by [project](/glossary#project), so you can see where your agent
time and spend go.

## Tools

The [tools](/glossary#tool) and [MCP servers](/glossary#mcp-server) your agents use
most, and the tools that return the most content back into the agent's context.

## Health

Signals about how your sessions go: interruptions, permission rejections, context
that grows too fast and how sessions end.
