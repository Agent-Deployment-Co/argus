# About

Argus is a free and open source tool from the team at
[The Agent Deployment Company](https://www.agentdeployment.co). It's
[MIT licensed](https://github.com/Agent-Deployment-Co/argus/blob/main/LICENSE),
so you're free to use, read and modify it.

## Why we built it

We build and deploy AI agents for go-to-market teams, and we kept hitting the
same wall: it's hard to see how people are actually using their agents, where
they get stuck and what's working. We built Argus to answer that, first for our
own clients and then for anyone. The goal is simple: help people get more out of
the agents they already use.

## The Agent Deployment Company

The Agent Deployment Company (ADC) helps go-to-market teams move from scattered
AI experiments to production agents that fit how they already work. Rather than
selling software, we embed engineers in a team's own environment and build agents
around the systems and data it already has. Argus grew out of that work, from
watching where real agent usage runs smoothly and where it doesn't.

Learn more at [agentdeployment.co](https://www.agentdeployment.co).
