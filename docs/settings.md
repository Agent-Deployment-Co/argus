# Settings

Configure Argus from inside the app. Open Settings from the left nav.

## What you can set

- API keys for the model providers Argus can use.
- Task interpretation, the optional pass that groups your work into tasks.
- Text retention, which controls whether Argus keeps the text of your prompts and
  responses on your machine.

For the full list of settings, including the ones you set on the command line, see
[Configuration](/configuration).
